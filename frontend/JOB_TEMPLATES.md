- [ ] ID is unique and higher than last job
- [ ] Title includes role and location/company
- [ ] Company is real Sri Lankan company
- [ ] Location is specific (city/area)
- [ ] Rate is in LKR/hr format
- [ ] Amount is reasonable (rate √ó hours)
- [ ] Type is Part-Time/Full-Time/Freelance/Contract
- [ ] Tags are 3-4 relevant keywords
- [ ] Logo color exists in Tailwind
- [ ] Description is 50-80 words
- [ ] Description includes: duty, benefit, flexibility
- [ ] postedBy is unique company ID
